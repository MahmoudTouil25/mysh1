# MYSH — Heavy Construction Equipment & Plants

Backend API for a platform dedicated to heavy construction equipment and plant machinery — built with **NestJS** and **TypeScript**.

## Stack

- [NestJS](https://nestjs.com/) v11
- TypeScript
- Node.js

## Setup

```bash
npm install
```

## Run

```bash
# development (watch mode)
npm run start:dev

# production
npm run start:prod
```

## Test

```bash
npm run test
npm run test:e2e
```

## Default port

`http://localhost:3000`
